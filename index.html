El proyecto trata sobre un sistema de créditos para un almacen de electrodomésticos, el cual está distribuido en tres partes.

1. Formulario para crear Clientes en la Base de datos del almacén

2. Asignar o rechazar el crédito

3. Pago del crédito(Revisar moras, realizar abonos a capital, formas de pago, entre otros.)
